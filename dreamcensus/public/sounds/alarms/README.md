# Alarm Sounds

This directory contains audio files for the alarm feature.

## Required Files

The following sound files need to be added:

1. **gentle-rise.mp3** (default)
   - Soft, gradual awakening sound
   - Duration: 30-60 seconds (will loop)
   - Format: MP3, 128kbps or higher

2. **morning-birds.mp3**
   - Natural birdsong
   - Duration: 30-60 seconds (will loop)
   - Format: MP3, 128kbps or higher

3. **dream-bells.mp3**
   - Mystical chimes matching app theme
   - Duration: 30-60 seconds (will loop)
   - Format: MP3, 128kbps or higher

4. **classic-chime.mp3**
   - Traditional alarm tone
   - Duration: 10-30 seconds (will loop)
   - Format: MP3, 128kbps or higher

5. **ocean-waves.mp3**
   - Gentle ocean sounds
   - Duration: 30-60 seconds (will loop)
   - Format: MP3, 128kbps or higher

## Audio Requirements

- **Format**: MP3 (best browser compatibility)
- **Bitrate**: 128kbps minimum
- **Sample Rate**: 44.1kHz
- **Channels**: Stereo or Mono
- **File Size**: Keep under 500KB per file for fast loading

## Sources

You can:
1. Create custom sounds
2. Use royalty-free audio from:
   - Freesound.org
   - Pixabay
   - YouTube Audio Library
   - Incompetech
3. Record your own

## Testing

Use the "Test" button in the alarm settings page to preview sounds before committing.
